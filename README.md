# Academic Identity Verifier (Japan)

**EN | 日本語**

---

## Overview

This project provides a public-information-based verification tool to assess whether a claimed academic or research identity has publicly verifiable existence in Japan.

The system does **not** determine fraud or authenticity.
It only evaluates the level of public existence support based on openly available sources.

---

## 概要（日本語）

本プロジェクトは、日本国内における学術・研究関連の身元について、
公開情報に基づき「存在性の裏付け」が確認できるかを検証するツールです。

本システムは、詐称や虚偽を断定するものではありません。
公開情報に基づく存在性の支持度を提示するのみです。

---

## Key Principles

- Public information only
- No definitive judgement
- Evidence-based output
- Compliance-first wording

---

## Supported Scope

- Region: Japan
- Target: Individuals claiming academic or research affiliation
- Language: English / Japanese

---

## What This Project Does NOT Do

- Does not assess academic quality
- Does not declare fraud or falsehood
- Does not use paid or private databases

---

## Architecture

- Backend: FastAPI (Python)
- Rule-based scoring engine
- Public data sources only (e.g. institutional directories, OpenAlex)
